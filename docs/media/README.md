# Capturas de pantalla

Esta carpeta contiene las capturas de pantalla para el README.md.

## Archivos necesarios

- `cli-demo.png` - Captura del CLI ejecutándose con `--help` o un envío exitoso

## Cómo generar las capturas

1. **CLI Demo:**
   ```powershell
   # Ejecutar y capturar
   python -m aeat_sender.cli --help
   # O
   python -m aeat_sender.cli --version
   ```
